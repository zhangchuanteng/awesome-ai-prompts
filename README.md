## ✍️ 通用写作 (15个)

**1. 万能写作助手** — 你是一位资深写作专家。请帮我写一篇关于 **[主题]** 的文章。要求：标题吸引人、结构清晰（开头引入→正文3-5点→结尾升华）、每点配案例、字数[字数]、风格[专业/轻松/学术/口语化]。

**2. 小红书爆款文案** — 写一篇小红书笔记，主题 **[主题]**。emoji开头标题含数字痛点词；开头3秒抓眼球；正文emoji分段每段≤3行；口语化像朋友聊天；结尾引导互动；5-8个话题标签；500-800字。

**3. 公众号深度文章** — 写一篇微信公众号文章，主题 **[主题]**。疑问句/数字型标题；故事/数据引入；3-5个小标题层层递进；观点配案例数据；结尾引导转发；2000-3000字。

**4. 文章改写润色** — 将以下文字改写为更通俗易懂的版本，保持核心观点不变。原文：[粘贴]。专业术语日常化、加生活类比、目标读者[描述]。

**5. 标题生成器** — 为主题 **[主题]** 生成10个不同风格标题：数字型/疑问型/对比型/揭秘型/情感型/权威型/紧迫型/故事型/清单型/反常识型。

**6. 中英翻译** — 将以下内容翻译为[目标语言]。自然地道、保留语气风格、专业术语准确。原文：[粘贴]

**7. 故事/小说创作** — 写一个[类型]故事。背景：[时间地点]；主角：[人物设定]；冲突：[核心矛盾]。开头抓人、节奏紧凑、对话生动、结尾出人意料。字数[字数]。

**8. SEO 文章** — 写一篇SEO优化文章，关键词 **[关键词]**。关键词密度2-3%；H2/H3合理分布；meta description 150字内；1500-2500字。

**9. 诗歌/歌词** — 以 **[主题]** 为主题创作一首[诗歌/歌词]。风格[豪放/婉约/现代/说唱]。押韵自然、意境优美。

**10. 演讲稿** — 写一篇[时长]分钟演讲稿，主题 **[主题]**，场合[会议/毕业典礼/年会]。开头金句、中间故事+观点、结尾号召。

**11. 读后感/书评** — 写一篇关于《[书名]》的读后感。概述+核心观点+个人感悟+推荐理由。800-1500字。

**12. 朋友圈文案** — 为主题 **[主题]** 写5条朋友圈文案。风格[文艺/搞笑/励志/日常]。简短有力、配图建议。

**13. 短视频脚本** — 写一个[时长]秒短视频脚本，主题 **[主题]**。前3秒钩子、中间干货、结尾引导关注。标注画面和台词。

**14. 产品说明书** — 为 **[产品名]** 写一份用户友好的说明书。功能：[功能列表]。通俗易懂、含FAQ。

**15. 行业报告摘要** — 总结以下行业报告核心要点：[粘贴报告]。输出：行业现状/核心趋势/关键数据/机遇挑战/行动建议。

## 💻 编程开发 (15个)

**16. 代码审查** — 审查以下代码：`[语言] [代码] `。从正确性/性能/安全性/可读性/SOLID原则审查，给出优化代码。

**17. SQL 优化** — 优化以下SQL：`sql [SQL] `。分析执行计划、索引建议、查询重写、给出优化后SQL。

**18. API 设计** — 设计RESTful API。业务：[场景]；实体：[列表]。输出：URL设计/统一响应/错误码/OpenAPI定义。

**19. 代码重构** — 重构：`[语言] [代码] `。遵循SOLID、消除异味、适当用设计模式、解释重构原因。

**20. 技术方案设计** — 设计 **[系统]** 技术方案。需求：[描述]；规模：[QPS]。含架构图/模块设计/数据库/时序图/性能优化/容灾方案。

**21. Bug 定位** — 现象：[描述]；环境：[信息]；代码：[代码]。分析最可能原因、排查步骤、修复方案。

**22. Git Commit 生成** — 根据变更生成规范commit message。遵循Conventional Commits：	ype(scope): description。

**23. 正则表达式** — 写出匹配以下规则的正则：[描述]。附带解释和测试用例。

**24. 代码翻译** — 将代码从 **[源语言]** 翻译到 **[目标语言]**，使用惯用写法。代码：`[代码] `

**25. 单元测试** — 为以下代码生成单元测试：`[语言] [代码] `。覆盖正常/边界/异常场景。

**26. 数据库设计** — 为 **[业务]** 设计数据库。实体：[列表]。输出ER图、建表语句、索引设计。

**27. Dockerfile** — 为 **[项目类型]** 编写Dockerfile。多阶段构建、镜像优化、安全最佳实践。

**28. 代码注释** — 为代码生成详细注释：`[语言] [代码] `。方法级文档+行内注释。

**29. 接口文档** — 为Controller生成接口文档：`[代码] `。URL/参数/响应/示例/错误码。Markdown格式。

**30. CI/CD 配置** — 为 **[项目]** 生成CI/CD配置。平台：[GitHub Actions/GitLab CI]。构建/测试/部署/通知。

## 🎨 AI 绘画 (6个)

**31. Midjourney 提示词** — 生成MJ提示词：[想法]。[主体],[背景],[风格],[光照],[构图],[画质] --ar [比例] --v 6

**32. SD 提示词** — 生成SD提示词。描述：[内容]。正向：主体+环境+风格+光照+画质。反向：避免的元素。

**33. Logo 设计** — 为 **[品牌]** 设计Logo。风格[极简/科技/复古]；色调[颜色]。3个方案+提示词。

**34. UI 设计灵感** — 为 **[产品]** 生成UI设计。风格[苹果风/毛玻璃/暗黑]。5个页面布局+配色HEX值。

**35. 图片描述转Prompt** — 将描述转为AI绘画提示词（MJ+SD格式）：[描述]。

**36. 视频分镜** — 为 **[视频主题]** 生成分镜脚本。每镜头：画面/景别/运镜/时长/配音/BGM。

## 📢 营销文案 (8个)

**37. 品牌故事** — 为 **[品牌]** 撰写品牌故事。英雄之旅结构，500字以内。

**38. 产品卖点(FAB)** — 提炼 **[产品]** 卖点。FAB法则，3-5个核心卖点+广告语。

**39. 社交媒体周计划** — 为 **[品牌]** 制定一周社交媒体计划。平台：[小红书/抖音]。每天1个主题+标题+标签。

**40. AIDA 营销文案** — 用AIDA为 **[产品]** 写文案。Attention→Interest→Desire→Action。200-400字。

**41. 活动策划** — 策划 **[活动类型]** 活动。含主题/流程/推广/预算/KPI。

**42. 竞品分析** — 对比 **[产品A]** vs **[产品B]**。功能/价格/体验/优劣势+选型建议。

**43. 电商详情页** — 为 **[产品]** 写详情页文案。主图标题→痛点→卖点→证言→售后。

**44. 邮件营销** — 为 **[活动]** 写营销邮件。吸引主题行/正文/CTA按钮文案。

## 📚 学习教育 (8个)

**45. 费曼学习法** — 解释 **[概念]**。10岁能懂的语言；2-3个类比；1个实例；2个误解；3个检验问题。

**46. 学习路线图** — 为 **[领域]** 构建路线。知识模块/学习顺序/资源推荐/阶段检验/常见陷阱。

**47. 读书笔记** — 《[书名]》笔记。一句话总结/核心观点/金句/感悟/行动清单/相关书籍。

**48. 论文解读** — 解读论文：[标题]。核心贡献/背景/方法创新/实验结果/局限性/启发。

**49. 知识点对比** — 对比 **[A]** vs **[B]**。定义/原理/应用/优缺点/代码示例。对比表格+选型建议。

**50. 考试备考** — 为 **[考试]** 制定备考计划。知识重点/每日计划/复习策略/模拟测试。

**51. 公开课笔记** — 整理课程笔记：[内容]。大纲/核心知识点/公式/示例代码/课后练习。

**52. 思维导图** — 为 **[主题]** 生成思维导图（Markdown格式）。核心概念展开2-3层。

## ⚡ 效率工具 (10个)

**53. 会议纪要** — 生成会议纪要：[记录]。主题/时间/要点/决议(责任人+截止日)/待办/遗留问题。

**54. 简历优化** — 优化简历。岗位：[JD]；简历：[内容]。STAR改写、量化成果、前后对比。

**55. PPT大纲** — 生成PPT大纲。主题：[主题]；时长：[时长]；听众：[画像]。每页标题+观点+视觉建议。

**56. 日报/周报** — 生成[日报/周报]。工作：[事项]。已完成/进行中/下一步。数据量化。日报≤100字/周报≤300字。

**57. 邮件撰写** — 写[类型]邮件。收件人：[职位]；目的：[诉求]。主题行+正文+行动号召。

**58. 待办清单** — 根据目标生成待办：[目标]。优先级排序、估算时间、标注依赖。

**59. SOP 流程** — 为 **[流程]** 编写SOP。步骤详细、操作说明、注意事项、常见问题。

**60. 项目复盘** — 复盘：[项目]。目标回顾/结果评估/做得好的/待改进的/教训/下一步。

**61. OKR 制定** — 为[岗位]制定Q[N] OKR。2-3个Objective，每个3-5个可量化Key Results。

**62. 时间管理** — 我的时间使用：[描述]。时间审计/优先级矩阵/日程模板/效率工具推荐。

**63. 数据分析报告** — 分析数据：[描述]。数据概览/关键发现/可视化建议/异常分析/结论建议。

**64. Python数据分析** — 用Python分析：[目标]。pandas+matplotlib，完整可运行代码。

**65. A/B测试方案** — 设计A/B测试。目标：[目标]；变量：[变化]。假设/样本量/分组/分析方法。

**66. 数据可视化建议** — 数据：[描述]。推荐可视化方式+选择理由+设计建议。

**67. SQL数据分析** — 用SQL分析：[需求]。表结构：[描述]。查询+解读+业务建议。

## 💼 职场沟通 (6个)

**68. 面试自我介绍** — 岗位：[岗位]；经历：[简述]。1-2分钟，背景→经历→技能→匹配度。

**69. 薪资谈判** — 当前：[薪资]；期望：[薪资]。话术/应对策略/兜底方案。

**70. 述职报告** — 岗位：[岗位]；周期：[时间]。概述→成果→亮点→成长→不足→计划。

**71. 离职沟通** — 原因：[原因]。得体表达/感谢/交接计划/保持关系。

**72. 向上汇报** — 内容：[内容]。结论先行→数据支撑→问题+方案→需要的支持。

**73. 工作交接文档** — 工作：[内容]。日常工作/待办/联系人/账号/注意事项/FAQ。

## 🎭 角色扮演 (5个)

**74. 模拟面试官** — [公司]面试官。岗位：[岗位]。一问一答+点评+综合评价+压力题。

**75. 辩论对手** — 辩论 **[话题]**，你站[对立面]。有论据、不人身攻击、最后总结。

**76. 头脑风暴** — brainstorm **[主题]**。约束：[限制]。每次5个想法，鼓励疯狂想法，筛选Top 3。

**77. 心理咨询师** — 温和倾听、共情理解、引导思考、开放式问题。

**78. 技术导师** — 经验丰富的技术导师。通俗解答、实际案例、学习资源推荐。

## 🧠 Prompt 技巧 (5个)

**79. 结构化Prompt** — # Role: [角色] | ## Profile | ## Skills | ## Rules | ## Workflow | ## Initialization

**80. Few-Shot** — 示例1：输入→输出；示例2：输入→输出。新任务：输入→

**81. Chain of Thought** — Step1明确问题→Step2列出因素→Step3逐一分析→Step4综合结论→Step5建议。

**82. 角色约束** — 你是[角色]。背景/性格/知识/风格。始终扮演，不打破第四面墙。

**83. Prompt优化器** — 优化我的prompt：[原始]。更精确指令/更好格式/添加示例/约束条件/输出要求。

## 🔧 生活助手 (6个)

**84. 旅行攻略** — [天数]天 **[目的地]** 旅行。预算：[预算]。每日行程/住宿/交通/美食/注意事项。

**85. 健身计划** — [天数]健身计划。目标：[减脂/增肌]；水平：[初/中/高级]。训练内容/饮食/休息。

**86. 菜谱推荐** — [数量]道[菜系]菜谱。食材常见、步骤详细、新手友好、含准备时间和难度。

**87. 选礼物** — 推荐[数量]个送[对象]的礼物。场合：[生日/节日]；预算：[范围]。含价格+理由+渠道。

**88. 理财规划** — 个人理财方案。月收入/支出：[金额]。预算分配/储蓄/投资方向/风险提示。

**89. 饮食计划** — 一周健康饮食。目标：[减重/增肌]；忌口：[列表]。每日三餐+加餐+简单做法。
